Tarantula is a variant on spider in which cards are considered only by colour, rather than by suit; this is sometimes known as two-suit spider. The aim is to sort the cards into stacks from king to ace in colour, and remove them from the game to the eight foundations. Cards can be stacked in rank order regardless of suit, but sequences can only be moved if all cards are of the same colour. When there are no more moves, a new card can be dealt on to the end of each of the ten piles.
