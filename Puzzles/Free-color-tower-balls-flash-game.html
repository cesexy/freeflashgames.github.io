Play Instructions:

The object of the game is really simple: the same coloured balls have to be arranged in one column. There are six columns of coloured balls, each column is a different colour. The tower is divided into seven rotating sections that enable you to move balls from column to column. To create a space in which to move balls there is one hole at the bottom level of the tower. For advanced Players: by the time you have become familiar with the game; step on the next level! Each ball is numbered from one to seven. On this level not only the coloured balls have to be arranged in one column but also the numbers should be put in order from one to seven (from top to bottom). Game time and moves are measured. If you want to be the BEST you have to solve this puzzle in the shortest time and with the fewest moves.
