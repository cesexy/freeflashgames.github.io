Play Instructions:

Help Kritaz unlock the treasures of the Puzzle Vaults by completing a series of devious and fun puzzles in a race against time. Start by clicking on the glowing arrow block to release new objects from the object queue (puzzle pieces, gems or bombs). Mouse Click on an object to select it. A highlight block shows the currently selected object. If a puzzle piece is selected the image on the right highlights where it should be placed on the game area. Use the keyboard ARROW keys to slide an object in a direction. An object will slide until it either hits a wall or another game object Slide the puzzle pieces around the game area to build up the picture of the completed puzzle as shown on the right hand side. The first time a piece is correctly placed it will glow white and points will be awarded. When open, use the 4 gates positioned around the game area to get rid of unwanted objects. Don't worry all objects will reappear later at the back of the object queue. The 'NEXT' area on the right displays what object will appear next from the object queue. So keep an eye on it and plan ahead. Match 3 or 4 gems of the same colour horizontally or vertically to gain bonus points. In a Gem level you will have to match 4 gems of a specific colour so be careful not to match 3 first or it will not be possible to get 4 as matched Gems do not return to the object queue. Use bombs to remove unwanted barrier blocks. Bombs will also remove puzzle pieces and gems but they will reappear at the back of the object queue (but beware so will the bombs). Bombs will explode after approx. 3 seconds but will not explode whilst moving so should give you plenty of time to get them into position.
