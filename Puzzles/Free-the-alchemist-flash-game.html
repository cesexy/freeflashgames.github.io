Game Description:

In this game there will be a grid of atoms of carbon, hydrogen and oxygen. You need to form molecules by arranging the atoms in a particular sequence. For example, one hydrogen, one oxygen and then one hydrogen will form one water molecule. There are 4 types of molecules that you can form: water, carbon dioxide, methane, and diamond. When the molecules are formed, the atoms will disappear and new atoms will fall from the top. In each level, you will need to create the specified number of each molecule. You lost the game if you cannot create the molecules within the time limit.
