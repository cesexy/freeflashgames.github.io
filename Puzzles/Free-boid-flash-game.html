Game Description:

Bringer Of Incremental Destruction You the player will gain control of a powerful celestial being (a B.O.I.D.) whom derives its greatest pleasure from destroying other celestial bodies, the more bodies you destroy simultaneously the more happiness points(score) you will accumulate.

Play Instructions:

The game functions as follows: When a new game is started there will appear a field containing 100 circular objects each containing a number in the range from zero to nine. When one clicks on an object it and its adjacent objects values will be increased by one. If an objects value is increased beyond the value nine the object will be destroyed, after the destruction of an object the players score will increase, how much the score is increased depends on how many objects that are destroyed simultaneously. If n is the number of destroyed objects the score increase is given by (2^n)*10.
