Game Description:

Just in time for Valentine's day! A popular variant on the match 3 theme. Use your cursor to swap blocks, you can set up chains by matching 3 or more blocks causing those blocks to explode, the blocks above fall, possibly causing a chain reaction, how long can you keep the chains going?

Play Instructions:

Players must match the same colored blocks to clear the blocks and score points. Clearing blocks causes the blocks above to fall, if those blocks cause additional clears then chains are formed and the player receives bonus points. Once a player is experienced the game becomes a race to create bigger chains to top the high score. Use your ARROW keys to move the cursor, the SPACE bar will swap the highlighted blocks. You can force the stack to raise up giving you more blocks to match on more quickly, using the SHIFT key. To activate the upgrades use the ENTER key. (The upgrades need to be unlocked.)
