Game Description:

The game is a sort of slot machine / puzzle game mix. The playfield is composed of a matrix of 9 rows and 6 columns. At regular time intervals a set of 6 blocks (showing one of 6 different seeds) is randomly generated at the lowest free row. Then the player has six seconds to swap the blocks by mouse click. When all the 9 rows are completed the game is over. The objective of the game is to line up the maximum number of same seed blocks either horizontally, vertically or diagonally to score the maximum number of points.
