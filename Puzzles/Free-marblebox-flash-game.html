Game Description:

MarbleBox is a challenging strategy game. Your objective is to mark all boxes that contains marbles. If you open a box that contains a marble, the game is over!

Play Instructions:

The objective of MarbleBox is to detect hidden marbles within a grid of boxes, while avoiding uncovering the actual marbles. The number in a cell shows how many marbles exist within the surrounding 8 cells. To win the game, you must correctly mark all the boxes that contain marbles, without actually opening any of those boxes. There are two difficulty levels. The easier one only deals with one marble per box, and the more difficult level has up to 3 marbles per box.
