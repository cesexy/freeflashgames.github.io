Minecraft Tower Defense is a tower defense game with ideas taken from mobs and items from Minecraft, by Mojang. Now, to work on it I would have to publish it after every edit, change some properties by hand outside of flash, and then run it, and repeat for every bug and feature I fix/find/add. Soooo..... Here is version 2.9, not quite 3.0. I'm now working on some other games, right now one called Ink Runner. I may eventually come back to this and remake it with more thrifty coding (I know a lot more now than I did when I started this project), but that will take a long time, so enjoy this version, which doesn't have potions, a shop, a storyline, lava/water, map-specific graphics, flint/steel, or multiplayer; all things I wanted to add.
