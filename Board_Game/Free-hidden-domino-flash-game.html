Game Description:

The game is also known as: Dominosa, Dominosa Omnibus, Solitaire Dominoes, Domino Hunt. Find and remove all hidden domino stones.

Play Instructions:

Click to select the two blocks (one domino stone) and press ENTER. If you selected one hidden stone it will be removed, if two - you lose one life. To start the game click on START LEVEL. To start next level click on NEXT LEVEL and next click on START LEVEL. To the right is a list of hidden stones. After the game, click on this list and see the place of the hidden stone.
