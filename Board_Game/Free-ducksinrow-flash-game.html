Game Description:

Ducks in Row is a game where you move tiles to line up three, four, or five birds in a row to score points. It's fun for the whole nest. Can you beat all nine levels?

Play Instructions:

Switch bird tiles by dragging a bird tile to an adjacent bird tile. Swapping tiles is only allowed if it makes a match, though. Empty tiles are filled from the bottom up, since birds fly up. Be careful, you only get a certain number of moves. If your move creates multiple matches, you create a combo and score more points.
